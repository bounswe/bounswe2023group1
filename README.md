# CMPE352 Group 1 -  Disaster Response Platform

## :page_with_curl: About The Project

Welcome to our Disaster Response Project Repository! We are a group of computer engineering students at Bogazici University, committed to building a software that can help improve the effectiveness and speed of disaster response efforts. Our shared goal is to make a real difference for the those in need during difficult times.

You can check out the [Wiki Page](https://github.com/bounswe/bounswe2023group1/wiki)  to get more information about us. You can also access team members' personal pages, meeting notes, research and more.

## 🏫 About The Course
This is the Fundamentals of Software Engineering course that aims to cover  main concepts, methods, and tools related to developing software products.
